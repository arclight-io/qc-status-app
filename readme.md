# QC-Status
QC-Status是一个去中心化的聊天室，采用Fediverse架构。
## 我该如何使用QC-Status？
### 对用户：
像使用中心化聊天室那样就好。Fediverse是Web2的去中心化方案，和中心化平台的用法相同。
### 对各个站长：
1，获取源码；\
2，使用SQL文件安装数据库；\
3，配置config/main_conf.php；\
4，在UCenter中添加该应用；\
5，将config/uc_conf.php修改为您的UCenter应用配置；\
6，修改config/portal/index.inc.html为您网站的门户；\
7，完成安装，开始使用吧！\
